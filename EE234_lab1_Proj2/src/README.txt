Control an RGB LED (LD4) as follows.
– Use the four switches (SW0 – SW3).
– Turn the other four switches (SW4 – SW7) off.
– If SW3 is OFF, LD4 is OFF. This has the highest priority.
– If SW3 is ON, LD4 is controlled as follows.
• Turn the R on/off if SW0 is ON/OFF.
• Turn the G on/off if SW1 is ON/OFF.
• Turn the B on/off if SW2 is ON/OFF
